# Excel Dashboard

This folder is intended for a manually built Excel dashboard based on the exports in `data/exports/`.

Recommended sheets:

- Dashboard
- Monthly Revenue
- Payment Mix
- Top Merchants
- Revenue by State
- Merchant Health
- At Risk Merchants
- Retention

Recommended Excel features:

- Pivot Tables
- Pivot Charts
- Slicers
- Timeline filters
- KPI cards
- Conditional formatting
