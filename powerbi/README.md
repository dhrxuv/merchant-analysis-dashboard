# Power BI Notes

This folder is reserved for `Dashboard.pbix`.

Suggested tabs for the Power BI file:

1. Executive Summary
2. Merchant Performance
3. Customer Retention
4. Delivery Operations
5. Reviews and CX

Recommended visuals:

- KPI cards for revenue, orders, AOV, repeat rate, cancellation rate
- Line chart for monthly revenue
- Bar chart for top merchants
- Map for state-level revenue
- Funnel or clustered bars for payment method mix
- Histogram or box plot for delivery delays
- Cohort heatmap for retention
